# 镜像服务简介

在学校的内网设立镜像服务器，从软件，系统的官方源拉取数据，并且提供一个在连接校内网络但不登陆哆点时的高速，大流量软件/系统下载的解决方案。

# 镜像资源概要

Nova的一台Raspberry Pi，外挂250Gib硬盘，接入中国教育网。

# 页面上方的xxx指纹是什么？

那是镜像站的SSL证书指纹，在浏览器中可以看到（各个浏览器查看位置因人而异，大多数浏览器在点击地址栏最左边的一个图标之后可以看到）

# 证书不被信任/出现错误？

一般只有校内用户会出现这种情况，由于我们的SSL证书是签发给域名（mirrors.lunaluna.org）的，而在校内未登录哆点时无法DNS，所以只能直接访问 IP（202.202.243.125），此时由于域名和IP不一样，所以会报个错。

出现证书错误不要惊惶，点击“查看证书”，手动比对我们提供的对应的xxx指纹，如果相互符合则可以安心地继续访问。
